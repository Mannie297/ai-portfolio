## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- Python 3.8+
- Git

### Installation
1. Clone the repository:
```bash
git clone https://github.com/yourusername/ai-portfolio.git
cd ai-portfolio
```

2. Install dependencies:
```bash
npm install
pip install -r requirements.txt
```

3. Set up environment variables:
```bash
cp .env.example .env
```
